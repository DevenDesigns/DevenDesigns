# DevenDesigns Full Project

This is a Next.js website for DevenDesigns.

## What's Included
- Country + modern style
- Custom logo section
- Wallpaper section
- Pricing
- Portfolio placeholders
- Reviews
- Order request form with upload field
- PayPal and Cash App payment info
- Mobile responsive design

## How to run locally
1. Install Node.js
2. Open this folder in a terminal
3. Run: npm install
4. Run: npm run dev

## How to deploy
1. Upload these files to GitHub
2. Import the repository into Vercel
3. Click Deploy
